# 386 VIN Decoder

Decode your VIN plate number using the 386 computer.

## Description

It seems Lindell dropped something from his desk, and surely he wouldn't miss it if you borrowed it indefinitely...
This mod adds a new diskette that you can use to decode your VIN plate by consulting the Corris company database directly.

## Usage

The typed VIN will be verified against Corris company database, so a dial-up connection is required for the query to go through.

1. Insert diskette
2. Navigate to removable media partition `a:`
3. Execute the program `vin`
4. You'll then be prompted to type the VIN number. See the image on: Corris Rivett VINPLATE decoder table﻿ on how to read the VIN plate.
5. If everything checks out you just have to press ENTER after the information download finishes to view the results.

IMPORTANT: During the query process, any connection problems will require you to make the request again.

## How to Install

1. Unpack the contents of the .zip file in the Mods folder
2. Done!
